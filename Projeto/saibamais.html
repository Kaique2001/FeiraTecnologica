Projeto Institucional para Feira Tecnológica de 2021

O nosso projeto é um sistema de carrinho de compras, desenvolvido com base no nosso tema escolhido para a Feira Tecnológica, que no caso é, Tecnologia e Sociedade no Século XXI, e o subtema: E-commerce.
O Nosso grupo é composto pelos seguintes alunos: Kaik silva, Kaique Dias, Rodrigo Oliveira, Guilherme Batista e Jean Luka. Somos todos estudantes atualmente do 2º Semestre (2/3) de Desenvolvimento de Sistemas da Etec da Zona Leste.

O código do nosso projeto se encontra no seguinte link do github: https://github.com/Kaique2001/FeiraTecnologica

*Obs: O arquivo de conexão que está presente no repositório se encontra com as credenciais vazias, caso queira rodar o projeto na sua máquina, crie a base de dados conforme o arquivo "loja_site.sql" indica e informe as credenciais de conexão do banco aonde você o criou para funcionar corretamente.

Gostaríamos de ressaltar que, esse projeto servira de base para o desenvolvimento do nosso TCC, aonde pretendemos realizar o desenvolvimento de um E-commerce completo desde o zero.

Nosso projeto foi totalmente desenvolvido em PHP e MySQL. Caso tenha interesse em saber o que cada um desenvolveu no projeto, saber como anda o desenvolvimento do nosso projeto final ou entrar em contato com algum de nós, segue abaixo nossos respectivos e-mails institucionais:
